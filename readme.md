## Command 
- `/start` - starting bot
- `/setlang` - setting current language
- `/ocrts` - tesseractjs ocr
- `/ocr` - ocr space
- `/tr` - translate text  
and any more.
## Features
- OCR
- Translate
- Duckbot Mata
- Anti Spam
- Kick,ban,mute  
and any more.

## Deploy 
- `git clone https://github.com/butthx/duckbot`
- `cd duckbot`
- `yarn install`
change `sample.env` to `.env` file and fill the env variable
- `yarn build` 
- `yarn start`
